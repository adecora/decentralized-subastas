# Sistema de Subastas Descentralizado

DApp completa para gestionar subastas en blockchain utilizando **Smart Contracts** en Solidity y frontend con **Next.js + React**. Permite crear subastas, realizar pujas, sistema de reembolsos para perdedores y confirmación de entrega. Desplegado en **BSC Testnet** con integración MetaMask.
